1. Document statuses facilitate organizing your work-flow. You define as many statuses as you need and call them what you wish to suit your requirements. Moreover, you can change the display text of a status at any point in tme without affecting your system. So if you do not particularly like the status description Archived and instead would like to call it Stored for example, you can do so at any point.

2. You do have to do some work to leverage this tool for better organization. You have to assign a status to each document when creating or editing a document record.
